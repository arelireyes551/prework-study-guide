HTML

  • Hypertext markup language (HTML): to give content structure within websites
  • HTML elements builds structure of webpaage and provide a framework that makes for easy navigation
  • common elements types used to describe and divide a page:
      • Meta-related elements
      Content division elements
      Semantic elements
  • HTML elements are composed of open & close tags represented with angle brackets "< >"
      • Two elements are especially important
      Head and Body

    Head '< head >'

  • Contains infor (Metadata)
  • Helps browser render page correctly
  • Not visible when user look at webpage

    • Body '< body >'

  • Contains content shown to user
  • elements we want users seeing & interacting with must be inside body

    • DOCTYPE '< DOCTYPE >'

  • Tells browser what document type it should expect when opened
  • Is a declaration
      • declarations ARE NOT case sensitive
      HTML elements ARE case sensitve

    HTML Lang = "En" '< html lang = "en" >'

  • Tells browser language of webpage
  • "en" means browser will be in English
      • An HTML Attribute defines additional info about an element
  • Always defined in beginning, never closing

    • Metadata Webpage

  • Uses to effectively serve our webpage to users
      • not necessarily seen by users but contains important info

    Title '< title >'

  • element defines title for a webpage

    • Style '< style >'
